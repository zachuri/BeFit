# BeFit Application

An application developed by ZACHURI and artifactz1 to be able to keep ourselves fit.
It will track three main categories we believe are essentail for body building.

- Weight
- Exercise
- Diet

## Stack

- Stack (T3-Stack)
  - [Next.js]
  - trpc
  - Tailwind CSS
  - Prisma Motion
  - Framer Motion
